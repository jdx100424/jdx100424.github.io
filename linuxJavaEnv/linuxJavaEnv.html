（1）下载地址

http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

（2）安装

在/usr/java路径，把安装包复制到这里，并执行tar -zxvf jdk-XXX-linux-x64.tar.gz

（3）环境变量配置 JAVA_HOME=/usr/java/jdk1.x.x_x

PATH=$JAVA_HOME/bin:$PATH CLASSPATH=$JAVA_HOME/jre/lib/ext:$JAVA_HOME/lib/tools.jar export PATH JAVA_HOME CLASSPATH

（4）刷新环境变量让其生效

刷新：source /etc/profile 验证：echo $JAVA_HOME

（5）java命令验证

java -version
